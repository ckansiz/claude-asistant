# Decisions Log

Live log. Every key decision during the sprint, one line, timestamped, with why. See `sprint` skill Step 2 for what counts as a "key decision".

**Format:**
```
[YYYY-MM-DD HH:MM] <agent>: <decision> — <why> (rejected: <alternatives, if any>)
```
