## Design Research Protocol
When doing UI/UX design research, produce a structured Design Brief:
- 6–10 reference sites with visual direction analysis
- What to avoid (AI-generic clichés, glassmorphism, gradient orbs)
- Typography + color + layout recommendations
- Turkish market specifics (iyzico, KVKK trust signals)

## Proven Reference Sites
- DoorLoop, Buildium — property management SaaS (trust-first patterns)
- ikas — Turkish SaaS (social proof, local trust signals)
- Linear — minimal dark hero, bold typography
- Vercel, Supabase — developer-facing dark themes
- Awwwards, Lapa.ninja — current UI trends
- Mobbin — mobile UI patterns

## Anti-Patterns (always call out)
- Glassmorphism hero cards
- Gradient orbs / mesh backgrounds
- "Revolutionary", "game-changing" copy
- Generic smiling-people stock photos
- AI-generic purple-blue palettes

## Turkish Market Notes
- Trust signals critical: iyzico, KVKK, PCI-DSS logos
- Customer count + testimonials with real names
- Freemium/try-before-buy expectation high
